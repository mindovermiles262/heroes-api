# Heroes API

A FastAPI Tutorial using SQLModel

https://sqlmodel.tiangolo.com/tutorial/fastapi/


## Startup

```
$ python3 -m venv venv
$ source venv/bin/activate
(venv)$ pip3 install -r requirements.txt
(venv)$ ./run.sh
```

## Testing

```
$ python3 -m venv venv
$ source venv/bin/activate
(venv)$ pip3 install -r requirements.txt
(venv)$ ./test.sh
```
